Integrate the patient catalogue

Every catalogue block can be embedded on its own — test grids per section or service, glossary, quiz, collection pricing, compliance, patient information, and more. Use the same UI as the patient catalogue on health.coleebri.com via Elementor HTML widgets, iframes, or a WordPress shortcode.

Local access: from the catalogue folder run python3 -m http.server 9876 (try 8765 if the port is busy). Open http://localhost:9876/integrate.html or http://localhost:9876/integrate/. Do not use file:// — embeds and asset paths will fail.

1. Host the catalogue files

Upload the catalogue directory to your server, for example:

https://health.coleebri.com/catalogue/

Must be served over HTTPS (not file://). Test embeds at:

…/catalogue/embed/?widget=glossary

Snippets below update when you change this URL.

6. WordPress / Elementor

      

        Paste an HTML widget on each category page — no plugin. See
        docs/WEBSITE-EMBED.md and
        integrate/elementor/coleebri-service-*.json.
      

      
<div data-coleebri-embed="tests"
  data-coleebri-base="https://health.coleebri.com/catalogue/"
  data-branding="none"
  data-site="https://health.coleebri.com/en"
  data-integrated="1"
  data-transparent="1"
  data-service="men"
  data-height="900"></div>
<script src="https://health.coleebri.com/catalogue/assets/coleebri-embed.js"
  data-base="https://health.coleebri.com/catalogue/"></script>

      

        Use the same base URL in data-coleebri-base and the script data-base.
        Enquire uses mailto:health@coleebri.com in integrated mode — use an Elementor form for a styled enquiry flow.

Notes

      
    
        
  • Iframes resize automatically when using the script embed (postMessage).
    • 
        
  • Elementor: use stretch section full width for wide glossary grids.
    • 
        
  • Glossary and marker check use Coleebri-written text plus outbound NHS/NICE links only (no copied official content).
    • 
        
  • Embed header presets control logo, name, and catalogue link separately. Default partner shows the logo only (no duplicate name) plus “Open full catalogue”. Use Hidden on health.coleebri.com or fine-tune with logo=0&name=1&link=1.
    • 
        
  • Section pages: widget=tests&category=profiles (or any SECTIONS id). Service pages: widget=tests&service=men (or any TRACKS id).
    • 
        
  • Filter test cards by one test (test=GH014), several (tests=GH014,GH042), or a group (group=most-ordered from ColeebriTestGroups in data.js).
    • 
        
  • widget=tabs renders category navigation that links to your hosted catalogue (set base URL above).
    • 
        
  • Coleebri home page: widget=most-ordered and widget=categories — header hidden automatically.
    • 
        
  • Do not embed via file:// — always use HTTPS on your live site.
